# ML from Scratch

A hands-on implementation of core machine learning algorithms from scratch, focused on building strong intuition and understanding of underlying mathematics without relying on high-level libraries.

---

## 🚀 Algorithms Covered

- Linear Regression
- Logistic Regression
- K-Nearest Neighbors (KNN)
- K-Means Clustering
- Principal Component Analysis (PCA)
- Neural Networks (from scratch)

---

## 🎯 Goals

- Understand ML algorithms at a fundamental level
- Implement models without using high-level libraries like scikit-learn
- Strengthen mathematical intuition
- Build a strong foundation for advanced ML & Deep Learning
